# Python Problems

## 初級

### 変数と出力
- **問題**: 変数「x」に「10」を代入し、その値を出力してください。
- **解答例**:
```python
x = 10
print(x)
```
- **解説**: 基本的な変数の代入と「print()」関数による出力です。

### 四則演算
- **問題**: 「3」と「5」を足した結果を出力してください。
- **解答例**:
```python
print(3 + 5)
```
- **解説**: 「+」演算子を使って数値の加算を行います。

### 文字列出力
- **問題**: 「Hello, Python」と出力してください。
- **解答例**:
```python
print("Hello, Python")
```
- **解説**: 文字列はシングルクォート「'」またはダブルクォート「"」で囲みます。

### 入力
- **問題**: 入力された文字列をそのまま出力してください。
- **解答例**:
```python
s = input()
print(s)
```
- **解説**: 組み込み関数「input()」を使うと、ユーザーからの入力を受け取ることができます。

### int変換
- **問題**: 入力された数値を整数に変換して出力してください。
- **解答例**:
```python
n = int(input())
print(n)
```
- **解説**: 「input()」で受け取った値は文字列となるため、「int()」で整数に変換します。

### 偶数判定
- **問題**: 入力された整数が偶数か奇数か判定し、「偶数」または「奇数」と出力してください。
- **解答例**:
```python
n = int(input())
print("偶数" if n % 2 == 0 else "奇数")
```
- **解説**: 剰余演算子「%」を使い、2で割った余りが0かどうかで判定します。

### for文
- **問題**: 0から4までの数字を順に出力してください。
- **解答例**:
```python
for i in range(5):
    print(i)
```
- **解説**: 「range(5)」は0から4までの整数を生成します。

### while文
- **問題**: 「while」文を使って、1から3までを順に出力してください。
- **解答例**:
```python
i = 1
while i <= 3:
    print(i)
    i += 1
```
- **解説**: 「while」文は条件がTrueである間、処理を繰り返します。

### リスト作成
- **問題**: リスト「[1, 2, 3]」を作成して出力してください。
- **解答例**:
```python
lst = [1, 2, 3]
print(lst)
```
- **解説**: リストは角括弧「[]」を使って定義します。

### リスト参照
- **問題**: リスト「[10, 20, 30]」の2番目の要素（20）を出力してください。
- **解答例**:
```python
lst = [10, 20, 30]
print(lst[1])
```
- **解説**: リストのインデックス（添字）は0から始まります。

### len関数
- **問題**: リスト「[1, 2, 3, 4]」の要素数を出力してください。
- **解答例**:
```python
print(len([1, 2, 3, 4]))
```
- **解説**: 組み込み関数「len()」を使うと、リストの長さ（要素数）を取得できます。

### 文字列長
- **問題**: 文字列「Python」の文字数を出力してください。
- **解答例**:
```python
print(len("Python"))
```
- **解説**: 「len()」関数はリストだけでなく、文字列の長さも取得できます。

### 関数定義
- **問題**: 受け取った引数をそのまま出力する関数「show」を作成してください。
- **解答例**:
```python
def show(x):
    print(x)
```
- **解説**: 関数は「def 関数名(引数):」で定義します。

### 関数呼び出し
- **問題**: 関数「show」に引数「5」を渡して呼び出してください。
- **解答例**:
```python
def show(x):
    print(x)
show(5)
```
- **解説**: 定義した関数は「関数名(値)」で実行（呼び出し）できます。

### 戻り値
- **問題**: 2つの数を足した結果を返す関数「add」を作成してください。
- **解答例**:
```python
def add(a, b):
    return a + b
```
- **解説**: 関数内で「return」を使うと、計算結果などを呼び出し元に返すことができます。

### コメント
- **問題**: コード内にコメントを書き、「ok」と出力するプログラムを作成してください。
- **解答例**:
```python
# コメント
print("ok")
```
- **解説**: Pythonでは「#」以降の文字はコメントとして扱われ、実行されません。

### 真偽値
- **問題**: 真偽値の「True」を出力してください。
- **解答例**:
```python
print(True)
```
- **解説**: Pythonには真を表す「True」と偽を表す「False」というブール型があります。

### 比較
- **問題**: 3が5より小さいかどうかを判定し、結果（True/False）を出力してください。
- **解答例**:
```python
print(3 < 5)
```
- **解説**: 比較演算子「<」などを使うと、関係性が正しいかどうかを真偽値で得られます。

### and演算
- **問題**: 「True and False」の評価結果を出力してください。
- **解答例**:
```python
print(True and False)
```
- **解説**: 論理演算子「and」は、両方がTrueの場合のみTrueになります。

### or演算
- **問題**: 「True or False」の評価結果を出力してください。
- **解答例**:
```python
print(True or False)
```
- **解説**: 論理演算子「or」は、少なくともどちらか一方がTrueであればTrueになります。

## 中級

### 辞書操作
- **問題**: 辞書「scores = {"Math": 80, "English": 70}」に「Science: 90」を追加し、「Math」の値を出力してください。
- **解答例**:
```python
scores = {"Math": 80, "English": 70}
scores["Science"] = 90
print(scores["Math"])
```
- **解説**: 辞書への追加は「辞書[新しいキー] = 値」、参照は「辞書[キー]」で行います。

### 内包表記
- **問題**: リスト内包表記を使って、1から5までの整数の平方（2乗）のリストを作成し出力してください。
- **解答例**:
```python
print([i*i for i in range(1, 6)])
```
- **解説**: リスト内包表記を使うと、for文を使ったリスト作成を1行で簡潔に記述できます。

### 条件付き関数
- **問題**: 引数 n が正の数なら「positive」、それ以外ならNoneを返す関数を作成してください。
- **解答例**:
```python
def check(n):
    return "positive" if n > 0 else None
```
- **解説**: 「値1 if 条件 else 値2」という三項演算子的な書き方をreturn文に使えます。

### enumerate
- **問題**: リスト「["a", "b", "c"]」を、インデックス番号付きで出力してください。
- **解答例**:
```python
for i, v in enumerate(["a", "b", "c"]):
    print(i, v)
```
- **解説**: 「enumerate()」関数を使うと、ループ処理時にインデックスと要素を同時に取得できます。

### upper
- **問題**: 文字列「python」をすべて大文字に変換して出力してください。
- **解答例**:
```python
print("python".upper())
```
- **解説**: 文字列メソッド「.upper()」は小文字を大文字に変換します。

### split
- **問題**: 文字列「a,b,c」をカンマで分割してリストとして出力してください。
- **解答例**:
```python
print("a,b,c".split(","))
```
- **解説**: 文字列メソッド「.split(区切り文字)」は、文字列を指定した文字で分割しリストにします。

### join
- **問題**: リスト「["a", "b", "c"]」の要素をハイフン「-」で結合して出力してください。
- **解答例**:
```python
print("-".join(["a", "b", "c"]))
```
- **解説**: 文字列メソッド「.join()」は、リストの要素を指定した区切り文字で結合します。

### 例外処理
- **問題**: 0での割り算（ゼロ除算）を行い、エラーが発生したら「error」と出力する処理を書いてください。
- **解答例**:
```python
try:
    1/0
except ZeroDivisionError:
    print("error")
```
- **解説**: 「try-except」構文を使うことで、予期せぬエラー（例外）が発生した際の処理を記述できます。

### lambda
- **問題**: 引数を2倍にして返す無名関数（lambda）を作成し、引数3で実行した結果を出力してください。
- **解答例**:
```python
f = lambda x: x * 2
print(f(3))
```
- **解説**: 「lambda 引数: 処理」と書くことで、名前のない関数を簡潔に定義できます。

### sorted
- **問題**: リスト「[3, 1, 2]」を昇順に並び替えた新しいリストを出力してください。
- **解答例**:
```python
print(sorted([3, 1, 2]))
```
- **解説**: 組み込み関数「sorted()」は、元のリストを変更せず、並び替えられた新しいリストを返します。

## 上級

### 再帰
- **問題**: 再帰呼び出しを用いて、階乗を計算する関数を作成してください。
- **解答例**:
```python
def fact(n):
    return 1 if n == 0 else n * fact(n-1)
```
- **解説**: 関数の中で自分自身を呼び出すことを「再帰」と呼びます。終了条件（ここではn==0）が必須です。

### ジェネレータ
- **問題**: 1から3までの数値を順に「yield」するジェネレータ関数を作成してください。
- **解答例**:
```python
def gen():
    for i in range(1, 4):
        yield i
```
- **解説**: 「yield」を使うと、値を一つずつ生成して一時停止するジェネレータ関数を作成できます。

### map
- **問題**: リスト「[1, 2, 3]」の全要素を2倍にしたリストを作成してください。（map関数を使用）
- **解答例**:
```python
print(list(map(lambda x: x*2, [1, 2, 3])))
```
- **解説**: 「map()」関数は、リストの全ての要素に対して指定した関数を適用します。

### filter
- **問題**: リスト「[1, 2, 3, 4]」から偶数だけを抽出したリストを作成してください。（filter関数を使用）
- **解答例**:
```python
print(list(filter(lambda x: x % 2 == 0, [1, 2, 3, 4])))
```
- **解説**: 「filter()」関数は、条件に合致する（Trueを返す）要素だけを取り出します。

### クラス
- **問題**: 名前（name）を属性に持つ「Person」クラスを作成してください。
- **解答例**:
```python
class Person:
    def __init__(self, name):
        self.name = name
```
- **解説**: クラスは「class」キーワードで定義し、「__init__」メソッドで初期化処理を行います。

### メソッド
- **問題**: 「Person」クラスに、名前を出力する「greet」メソッドを追加してください。
- **解答例**:
```python
class Person:
    def __init__(self, name):
        self.name = name
    def greet(self):
        print(self.name)
```
- **解説**: クラス内に定義された関数を「メソッド」と呼び、第一引数（通常self）でインスタンス自身を受け取ります。

### 継承
- **問題**: 「Person」クラスを継承した「Student」クラスを作成してください。
- **解答例**:
```python
class Student(Person):
    pass
```
- **解説**: 「class 子クラス(親クラス):」と書くことで、親クラスの機能を継承できます。

### ファイル書込
- **問題**: 「test.txt」というファイルを作成し、「hello」と書き込んでください。（with文を使用）
- **解答例**:
```python
with open("test.txt", "w") as f:
    f.write("hello")
```
- **解説**: 「with open(...)」を使うと、ファイル操作後に自動的にクローズ処理が行われます。

### ファイル読込
- **問題**: 「test.txt」を開いて、内容を読み込み出力してください。
- **解答例**:
```python
with open("test.txt") as f:
    print(f.read())
```
- **解説**: 読み込みモード（デフォルト）でファイルを開き、「.read()」で内容を取得します。

### random
- **問題**: 「random」モジュールを使って、1から6までのランダムな整数を出力してください。
- **解答例**:
```python
import random
print(random.randint(1, 6))
```
- **解説**: 標準ライブラリの「random」を使うことで、乱数生成などの機能を利用できます。
