# プログラミング学習支援AIシステム

このシステムは、Gemini APIを利用してユーザーが提出したプログラミングコードを評価・フィードバックする学習支援ツールです。

## 事前準備

 **Gemini APIキーの取得**
   - [Google AI Studio](https://aistudio.google.com/) にアクセスし、APIキーを生成してください。<br>
   参考：https://qiita.com/IshigiwaKenichiro/items/3da8fb4f3bb2e724fe97
   - GeminiAPIキーの無料枠を使うため、料金は発生しません。

## Docker環境の構築

以下のサイトに従ってDocker Desktopをダウンロードしてください。<br><br>
https://qiita.com/zembutsu/items/a98f6f25ef47c04893b3<br><br>
すでにダウンロードしている方は、次に進んで下さい。

## GitHubからコードを取得する
1.ページ上部の 「Code」ボタン をクリックします。

2.表示されたメニューから 「Download ZIP」 を選択します。

3.ZIPファイルがダウンロードされるので、解凍して使用してください。

## 起動方法

1. **設定ファイルの準備**:
   - ZIPファイルを任意の場所に解凍して下さい。（Desktopなど）
   - 解凍したフォルダにある `.env.example` をコピーして `.env` という名前に変更します。もしくは、`.env.example` を `.env` という名前に変更して書き込んでもらっても構いません。
   - `.env` ファイルをテキストエディタで開き、`GEMINI_API_KEY=your_api_key_here` の部分を取得したAPIキーに書き換えて保存してください。

2. **ターミナル（PowerShellやコマンドプロンプト）を開く**: 
   - 解凍したファイルの位置に移動します。（解答場所がDesktopの場合）
   ```bash
   cd C:\Users\user\Desktop\programming-learning-system-gemini
   ```
3. **起動コマンドを実行**:
   - Docker Desktopを起動してから、ターミナルで以下のコマンドを実行してください。
   
   ```bash
   docker-compose up -d
   ```
   ※初めて起動する際は、自動的にイメージのダウンロードが始まります。
4. **ブラウザでアクセス**:
   - [http://localhost:5000](http://localhost:5000) にアクセスしてください。

## 終了方法

ターミナルで以下のコマンドを実行してください。
```bash
docker-compose down
```

## 使い方

1. **ログイン**: 「ユーザー名」はuser、「パスワード」はuserpassでログインしてください。
2. **問題の出題**: チャレンジモードと練習モードがあります。
   - チャレンジモード: 初級問題から始まり、点数を稼ぐことで問題がレベルアップします。このモードは必ずしてもらいたいです。
   - 練習モード: 既存の問題を選び、練習できます。
3. **コードの提出**: エディタにコードを書き、「回答を提出」ボタンを押すと、AIによる採点とフィードバックが表示されます。
4. **学習履歴**: 提出したコードの履歴を確認できます。ここで「CSVで保存」を押すと、提出履歴をCSVファイルとして保存できます。
   - csvファイルは「p93m8ein@s.okayama-u.ac.jp」宛に送ってもらえると幸いです。
5. **アンケート**: メールに添付してあるアンケートフォームに回答してください。

## トラブルシューティング

**Docker起動時にエラーが出る場合**
- Docker Desktopが起動しているか確認: スタートメニューから「Docker Desktop」を実行し、クジラのアイコンが安定するまで待ってから再度コマンドを打ってください。
